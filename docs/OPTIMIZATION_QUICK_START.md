# 提示词优化功能快速上手

## 🎯 3 步开始使用

### 第 1 步：打开编辑器

1. 在提示词列表页面，点击任意提示词卡片
2. 进入提示词编辑页面

> ⚠️ **重要**：新建提示词时无优化功能，需先保存后再编辑

### 第 2 步：找到优化按钮

在编辑页面，找到"提示词内容"输入框，你会看到：

```
┌─────────────────────────────────────────────────────┐
│ 提示词内容 *        📊 250 tokens   [AI 优化 ⚡]    │
│ ─────────────────────────────────────────────────── │
│                                                     │
│  [在这里输入或编辑提示词内容...]                      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

**"AI 优化"** 按钮位于：
- 📍 位置：内容输入框的**右上角**
- 🎨 样式：紫色渐变按钮，带闪电图标 ⚡
- 📏 大小：紧凑型按钮，不占用太多空间

### 第 3 步：选择场景并优化

点击 **"AI 优化"** 按钮，会弹出 5 个场景选项：

```
┌─────────────────────────┐
│ ✨ 通用优化             │  ← 不确定时选这个
├─────────────────────────┤
│ ✍️ 内容创作             │  ← 文章、文案
├─────────────────────────┤
│ 💻 代码生成             │  ← 编程相关
├─────────────────────────┤
│ 📊 数据分析             │  ← 数据处理
├─────────────────────────┤
│ 💬 对话交互             │  ← 聊天机器人
└─────────────────────────┘
```

**选择一个场景**，然后：
1. 按钮变为 "优化中..." ⏳
2. 等待 3-10 秒
3. 看到成功提示 ✅

```
┌──────────────────────────────┐
│ ✅ 优化完成！                │
│ Token: 280 | 成本: $0.0012   │
└──────────────────────────────┘
```

4. **内容已自动更新**到输入框中！

### 第 4 步：保存或撤销

**满意优化结果？**
- ✅ 点击底部 **"保存更改"** 按钮

**不满意？**
- 🔄 按 `Ctrl+Z` (Windows) 或 `Cmd+Z` (Mac) 撤销
- 🔄 或选择其他场景重新优化
- 🔄 或点击 **"取消"** 放弃更改

## ⚠️ 常见问题

### 找不到优化按钮？

**检查清单**：
- [ ] 是否在**编辑模式**而非新建模式？
- [ ] 提示词是否已经**保存**到数据库？
- [ ] 浏览器窗口是否足够宽？（窄屏可能隐藏按钮）

**解决方案**：
1. 如果是新建提示词，先点击 **"创建提示词"**
2. 然后从列表中找到该提示词，点击进入编辑
3. 现在就能看到 **"AI 优化"** 按钮了

### 点击按钮没反应？

**可能原因**：
1. 提示词内容为空
2. 网络连接问题
3. OpenAI API 配置错误

**快速检查**：
1. 确保内容框中有文字
2. 检查网络连接
3. 查看浏览器控制台是否有错误信息

### 优化结果不理想？

**解决方案**：
1. **立即撤销**：`Ctrl+Z` / `Cmd+Z`
2. **换个场景**：选择更合适的优化场景
3. **手动调整**：在优化基础上手动修改
4. **版本回滚**：从版本历史中恢复原版本
